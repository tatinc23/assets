# TAT Inc — Public Assets

Public media hosting for TAT Inc projects. GIFs, images, and other assets that need publicly accessible raw URLs.

## Structure

```
gifs/
  retro-blaster.gif      — retroblaster.tatinc.us gameplay
  karen-bot-hook.gif     — Karen Bot catching rm -rf demo
  karen-bot-story.gif    — Karen Bot 3-strike enforcement story
```

## Usage

Reference files via raw GitHub URL:
```
https://raw.githubusercontent.com/tatinc23/assets/main/gifs/<filename>
```
